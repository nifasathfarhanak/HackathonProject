# Estimated Implementation Cost

| Cost Category             | Description                                                     | Estimated Cost                                                                        |
| :------------------------ | :-------------------------------------------------------------- | :------------------------------------------------------------------------------------ |
| **1. Personnel (One-Time)** | The development team's time to build and deploy all phases.     | **~85-95% of Total Cost.** Estimated at **7-11 Sprints** for a small team.              |
| **2. Cloud Services (Ongoing)** | Pay-as-you-go for Google Cloud services (AI, Hosting).          | **Low.** Likely **<$50/month** during development and initial rollout due to generous free tiers. |
| **3. Maintenance (Ongoing)**  | Bug fixes, security updates, and monitoring.                    | **~15-20%** of the initial personnel cost, per year.                                    |
